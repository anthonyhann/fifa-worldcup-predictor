# 贝叶斯分析框架参考

## 核心理念

贝叶斯推断在足球预测中的应用核心是将"先验信念"（赛前基础实力评估）与"新证据"（伤停、首发、战术克制等）结合，产出"后验概率"。

## 分析链路

```
P0(胜/平/负) → 战术语境修正 → 联赛特性修正 → 似然更新 → P1(胜/平/负 | 数据)
     ↑                                                              ↓
  Elo + 泊松                                                   剧本预测
```

## A) 先验构建 (P0)

1. **Elo评级**：从 `data/elo_ratings.json` 读取，使用标准Elo期望公式
2. **泊松模型**：基于球队场均进球/失球（`data/team_stats.json`），计算预期进球xG
3. **加权融合**：Elo权重(默认30%) + 泊松权重(默认70%)
4. **主场优势**：从 `data/corrections.json` 读取home_advantage因子(+5%)

## B) 战术语境修正

### 控球陷阱检测
- 若场均控球率 > 60% 且 Field Tilt < 50% → 伪强队风险
- 世界杯场景下，可通过弱队防守系数间接识别

### 风格克制
- 铁桶阵克攻强守弱：弱队失球率<0.9 vs 强队进球率>1.8 → +4%爆冷
- 反击型克控球型：弱队进球率>1.3且失球率<1.0 vs 强队进球率>2.0 → +3%爆冷

## C) 联赛/赛事特性修正

世界杯特有修正项：
- 首轮不确定性(+3%)
- 48队扩军红利(+3%)
- 北美夏季高温/长途旅行(+2%)
- 卫冕冠军低谷(-2%)
- 东道主揭幕战加成(+8%)

## D) 似然更新 (P0 → P1)

通过联网搜索获取最新信息更新先验：
- 伤停/停赛：核心球员缺席(-3%~-5%)
- 预计首发：中轴线完整性检查
- 赛程密度：n天m赛
- 临场新闻：主帅发布会、训练消息

## 输出要求

每次分析必须包含：
1. P0(胜/平/负) 和 P1(胜/平/负|数据) 对比
2. 至少2套剧本预测
3. 至少3条风险提示
4. 数据来源与时间戳
